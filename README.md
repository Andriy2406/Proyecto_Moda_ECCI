¿De qué trata este proyecto?

Sitio web académico desarrollado como parte de la formación FrontEnd en ADSO. El sitio documenta la participación en el evento **Moda ECCI 2026**, donde la diseñadora **Sara Viloria** dictó una conferencia sobre teoría del color, semiología visual y el sistema Pantone.

El objetivo fue conectar lo aprendido en diseño visual con las competencias de desarrollo FrontEnd, construyendo un producto digital funcional, bilingüe y responsive.


Páginas del sitio

| Archivo | Contenido |
|---|---|
| `index.html` | Página de inicio: presentación, objetivo del sitio y conferencista |
| `experiencia.html` | Galería de fotos, aprendizajes en accordion y reflexión personal |
| `glosario.html` | 15 términos técnicos en inglés con traducción y definición |
| `sostenibilidad.html` | Economía circular, reflexión sobre tecnología y moda sostenible |
| `referencias.html` | Herramientas de IA usadas, tecnologías del proyecto y versículo bíblico |



Tecnologías utilizadas

- **HTML5** — Estructura semántica
- **CSS3** — Estilos propios con diseño limpio y responsive
- **JavaScript** — Carrusel de fotos, cambio de idioma ES/EN, menú móvil
- **Bootstrap 5.3** — Navbar, Cards, Accordion, Modal, Grid System, Alerts, Buttons

Componentes Bootstrap incluidos

- Navbar responsive con hamburguesa en móvil
- Grid System (mini tarjetas en index y sostenibilidad)
- Cards (sección de moda y software en experiencia)
- Carousel (carrusel propio en JS, no el de Bootstrap)
- Accordion (aprendizajes en experiencia y preguntas en sostenibilidad)
- Modal (reflexión personal completa)
- Alert (alerta informativa en index)
- Buttons (navegación y modal)
